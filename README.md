# SkillScope

AI powered Resume Analyzer and Interview Generator.

## Tech Stack

Frontend
- React
- Vite
- SCSS

Backend
- Node.js
- Express

AI
- Google Gemini API

Database
- MongoDB

## Features

- Resume upload
- AI skill extraction
- Match score with job description
- Interview question generation
- PDF report generation

## Installation

Backend
npm install
npm run dev

Frontend
npm install
npm run dev
